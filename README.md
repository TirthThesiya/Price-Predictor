# Price Predictor

*In this project we have integrated 3 prediction projects*

**1) Car Price Predictor**
**2) Airplane Fare Predictor**
**3) Laptop Price Predictor**

*Techonologies used :*
1) Python
2) Streamlit
3) Linear Regression
4) Random Forest
5) Pickel
6) Jupyter Notebook
7) Scikit Learn
8) Pandas
9) Matplotlib
10) Seaborn
